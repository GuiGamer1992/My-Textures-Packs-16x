# Lucid PvP 16x

Um pacote de textura PvP 16x limpo e compatível com FPS para Minecraft 1.8.9, projetado para Bedwars e jogabilidade competitiva.

## Características

- Resolução 16x
- GUI limpa
- Compatível com FPS
- Experiência PvP suave
- Estilo Espada Curta
- Otimizado para Bedwars

## Instalação

1. Baixe o arquivo `.zip`
2. Abrir Minecraft
3. Ir para:
   Opções → Pacotes de recursos → Abrir pasta de pacotes
4. Mova o arquivo `.zip` para a pasta
5. Habilitar o pacote no Minecraft

## Compatibilidade

- Edição Java do Minecraft
- Versão: 1.8.9
